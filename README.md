# RPG Hotel Dashboard Skeleton

This project now includes:

- Multi-hotel dashboard page (`/`)
- Admin page to add/remove hotels and event types (`/admin`)
- JSON persistence in `data/store.json`

## Run

```powershell
node .\server.js
```

Then open:

- `http://127.0.0.1:3000/`
- `http://127.0.0.1:3000/admin`
